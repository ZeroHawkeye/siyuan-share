# 思源分享插件（SiYuan Share）

> 将选择的思源笔记文档或块，快速发布为公网可访问的阅读页面，便于临时演示、协作查看或对外分享。

## 功能概述

- 一键分享：在笔记中选择文档/块后即可生成分享链接。
- 增量更新：重复分享时仅更新变更内容（视版本实现情况）。
- 资源处理：自动携带图片、附件等静态资源。
- 链接管理：可在插件面板查看、撤销已发布的分享。
- 基础访问统计（规划中）。

## 服务端配置

- 测试服务端地址：<https://share.sec-lab.cn>
  - 仅供测试，数据与稳定性不作保证，请勿存放重要/敏感信息。
- 在插件“设置”面板中可自定义服务端地址及相关参数。

## 快速使用

1. 在思源集市安装并启用插件。
2. 打开需要分享的文档（或定位到某块）。
3. 使用插件提供的分享入口（按钮/菜单）。
4. 首次使用在设置中填写服务端地址（可用测试地址或自建）。
5. 得到分享链接，复制给需要查看的人。
6. 如需撤销，在“分享列表”中删除对应记录即可失效。

## 常见问题

| 问题 | 说明 |
|------|------|
| 链接无法访问 | 检查服务端地址是否可用或是否已撤销分享 |
| 图片不显示 | 可能资源上传未完成，稍后刷新或重新分享 |
| 测试地址数据丢失 | 测试环境会不定期清理，请改用自建稳定服务端 |
| 如何提升隐私 | 避免分享含敏感信息的文档，撤销后链接不可继续访问 |
