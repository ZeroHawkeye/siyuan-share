# 静态资源管理 - 快速使用指南

## 访问资源管理界面

1. 打开思源笔记
2. 进入 **设置 → 集市 → 已下载 → 插件 → 分享插件**
3. 点击 **⚙️ 设置** 图标
4. 在设置面板中找到 **"静态资源管理"** 按钮
5. 点击按钮打开资源管理界面

## 界面说明

### 顶部区域

- **标题栏**: 显示"静态资源管理"和资源统计信息
- **搜索框**: 输入文件名关键词进行搜索
- **文档筛选**: 下拉选择特定文档
- **清除筛选**: 重置所有筛选条件

### 资源卡片

每个资源显示以下信息：

- **预览区**: 图片显示缩略图，其他文件显示类型图标
- **文件名**: 完整文件名（悬停显示完整路径）
- **文件大小**: 自动转换单位（KB/MB/GB）
- **所属文档**: 关联的思源文档标题
- **上传时间**: 资源上传到 S3 的时间
- **S3 键名**: 存储在 S3 的完整路径

### 操作按钮

- **👁️ 预览**: 查看资源内容
  - 图片：弹窗显示大图
  - 视频：嵌入播放器
  - 音频：嵌入播放器
  - 其他：浏览器新标签页打开

- **📋 复制链接**: 复制 S3 访问 URL

- **🔗 在浏览器打开**: 新标签页打开资源

## 使用示例

### 示例 1: 查找特定图片

```
1. 在搜索框输入: "screenshot"
2. 列表实时显示所有包含 "screenshot" 的文件
3. 点击预览按钮查看图片
```

### 示例 2: 查看某个文档的所有资源

```
1. 点击"文档筛选"下拉框
2. 选择目标文档（如"用户指南"）
3. 列表只显示该文档的相关资源
```

### 示例 3: 复制资源链接用于外部分享

```
1. 找到需要分享的资源
2. 点击"复制链接"按钮
3. 提示"已复制到剪贴板"
4. 在其他地方粘贴使用该链接
```

### 示例 4: 批量查看视频资源

```
1. 在搜索框输入: ".mp4"
2. 列表显示所有 MP4 视频文件
3. 逐个点击预览按钮观看
```

## 提示与技巧

### 搜索技巧

- **文件名搜索**: 支持部分匹配，不区分大小写
- **扩展名搜索**: 输入 `.jpg` 查找所有 JPG 图片
- **日期搜索**: 如果文件名包含日期，可以搜索日期字符串

### 筛选组合

可以同时使用搜索和文档筛选：

```
1. 选择文档: "项目文档"
2. 输入搜索: "diagram"
3. 结果: 项目文档中所有包含 "diagram" 的文件
```

### 快速清除

点击"清除筛选"按钮可一键重置：
- 搜索框内容
- 文档选择
- 显示全部资源

### 资源统计

顶部统计信息实时更新：
- 当前筛选条件下的资源数量
- 资源总大小（自动换算单位）

## 常见问题

### Q: 为什么列表是空的？

**A**: 可能的原因：
1. 尚未启用 S3 存储功能
2. 没有分享过包含资源的文档
3. 所有分享都使用了原始内容（未启用 S3）

**解决方法**: 
- 在设置中启用 S3 存储
- 分享一个包含图片的文档

### Q: 资源显示的文档名是 ID？

**A**: 当文档已被删除或重命名时，可能显示文档 ID。

**解决方法**: 
- 这不影响资源访问
- 可以根据上传时间判断来源

### Q: 预览失败怎么办？

**A**: 可能的原因：
1. S3 存储桶权限设置问题
2. 资源已被删除
3. 网络连接问题

**解决方法**: 
- 使用"在浏览器打开"按钮直接访问
- 检查 S3 存储桶访问权限

### Q: 如何删除资源？

**A**: 当前版本暂不支持直接删除。

**计划功能**: 
- 批量删除
- 按条件删除过期资源
- 存储空间优化

## 最佳实践

### 1. 定期检查资源

建议每月检查一次：
- 查看存储空间使用情况
- 清理不再需要的文档资源
- 验证重要资源的可访问性

### 2. 使用描述性文件名

上传资源时使用清晰的文件名：
- ✅ `project-diagram-2025-11.png`
- ❌ `IMG_1234.png`

便于后续搜索和管理。

### 3. 合理组织文档

为不同类型的内容创建不同文档：
- 项目文档
- 技术文档
- 设计资源
- 临时分享

便于按文档筛选资源。

### 4. 备份重要资源

对于重要的共享资源：
- 复制链接保存到安全位置
- 定期验证链接可用性
- 考虑在本地保留备份
